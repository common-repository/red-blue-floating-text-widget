An HTML/text widget that sticks to the top of the screen when user scrolls the window. 

== Description ==

This is a text widget that sticks to the top of the screen when user scrolls past it. The widget is designed so that it correctly "pops" out of the sidebar and slides over all other widgets that are below it. 

Multiple scroll widgets are supported as well. 

== Installation ==

Upload the plugin ZIP into wordpress and then activate the plugin. 

If you are using FTP then upload the whole "redbluefloater" directory into your wp-content/plugins folder.
